# Cursor Skills

Project skills live in this directory. Each skill is a folder with a `SKILL.md` file.

## Structure

```
.cursor/skills/
└── your-skill-name/
    └── SKILL.md
```

## SKILL.md example

```markdown
---
name: your-skill-name
description: When to use this skill (agent matches by description).
---

Instructions for the agent go here.
```

Add new skills as sibling folders next to this README.
